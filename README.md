# LatestNews

Below are the tools currently we have used.

<b> Kotlin</b></br>
<b> MVVM architecture</br>
<b> Dagger</br>
<b> Retrofit</br>
<b> Databinding</br>
<b> RxJava</br>
<b> Unit test(Junit, Mockito,PowerMockito)</br>
<b> Room(Database persistence)</br>
<b> Glide(To render image using URL)</br>
